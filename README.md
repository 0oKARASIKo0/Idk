# Qwen AI Agent

Автономный AI-агент с непрерывным мышлением, управлением мышью/клавиатурой, голосовыми командами, камерой и текстом в речь.

## Особенности

### 🧠 Непрерывное мышление
Агент думает **постоянно**, а не только когда вы что-то говорите. Он анализирует окружение, принимает решения и готов действовать в любой момент.

### 🎤 Голосовые команды
- Говорите в любое время - агент услышит и поймёт
- Распознавание команд на русском и английском
- Агент может изменить стратегию на основе ваших слов

### 📷 Компьютерное зрение
- Постоянный анализ камеры
- Обнаружение движения
- Распознавание объектов и сцен

### 🖱️ Управление мышью и клавиатурой
- Перемещение мыши
- Клики и перетаскивание
- Ввод текста
- Горячие клавиши

### 🔊 Текст в речь
- Агент говорит свои мысли и действия
- Русскоязычный голос
- Очередь сообщений

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Скачайте локальную LLM модель (ОБЯЗАТЕЛЬНО):
```bash
download_model.bat
```

Рекомендуемая модель: **Qwen 3.5 3B** (6.5GB, новейшая, 3 млрд параметров)

Доступные модели:
- `Qwen 3 0.6B` - 1.3GB - Быстрая, русский
- `Qwen 3 3B` - 6.0GB - Хороший баланс, русский
- `Qwen 3.5 3B` - 6.5GB - НОВЕЙШАЯ, лучшая, 3B params (РЕКОМЕНДУЕТСЯ)
- `Qwen 3.5 7B` - 14GB - Максимум качества, нужен GPU
- `Qwen 2.5 3B` - 6.0GB - Стабильная, русский

3. Запустите агента:
```bash
start_agent_admin.bat
```

## Команды в интерактивном режиме

- `status` - Показать статус агента
- `stop` - Остановить агента
- `start` - Запустить агента
- `quit` - Выйти из программы
- Любой другой текст - будет добавлен в контекст агента

## Голосовые команды

Агент распознаёт команды даже во время работы:

- **"стоп" / "stop"** - Остановить агента
- **"помощь" / "help"** - Показать возможности
- **"что делаешь?"** - Статус агента
- **"измени стратегию"** - Сменить стратегию
- **"наблюдай" / "смотри"** - Режим наблюдения
- **"анализируй"** - Режим анализа
- **"жди"** - Режим ожидания

Любая другая речь будет автоматически добавлена в контекст мышления агента.

## Как это работает

### Локальная LLM

Агент использует модель от HuggingFace:

1. Запустите `download_model.bat` для скачивания модели
2. Модель сохраняется в `~/.cache/qwen_agent/models/`
3. При запуске агент автоматически загружает модель
4. Модель работает на CPU (медленнее) или GPU (быстрее)

Приоритет LLM:
1. **Локальная модель** (HuggingFace) - работает оффлайн
2. **Remote API** (Ollama/remote сервер) - если настроен
3. **Offline режим** (правила) - если нет LLM

Это даёт:
- **Полную приватность** - данные не уходят с компьютера
- **Бесплатное использование** - нет платы за API
- **Работу без интернета** - после скачивания модели
- **Любую модель** - можно сменить через `download_model.bat`

### Непрерывное мышление

```python
while thinking:
    context = build_context()  # Собрать контекст из всех источников
    thought = generate_thought(context)  # Подумать
    action = decide_action(thought)  # Решить что делать
    execute_action(action)  # Действовать
    sleep(2 seconds)  # Пауза перед следующим циклом
```

Агент циклически:
1. Собирает данные от всех модулей (камера, голос, экран)
2. Анализирует ситуацию
3. Формирует мысль
4. Принимает решение о действии
5. Выполняет действие (если нужно)
6. Обновляет память

### Обработка голоса

Голос обрабатывается **асинхронно**:
- Агент не ждёт ввода голоса
- Когда вы говорите, текст добавляется в контекст
- Агент сам решает, как реагировать
- Специальные команды могут изменить стратегию

### Стратегии

Агент может переключать стратегии:
- `observe_and_assist` - Наблюдать и помогать
- `active_assistance` - Активная помощь
- `analysis` - Глубокий анализ
- `waiting` - Ожидание
- `conversation` - Разговор

## Устранение проблем

### Камера не работает
- Проверьте индекс камеры (CAMERA_INDEX=0, 1, 2...)
- Убедитесь что камера не используется другой программой

### Голос не распознаётся
- Проверьте микрофон в системе
- Уменьшите VOICE_ENERGY_THRESHOLD
- Говорите громче и ближе к микрофону

### Мышь не двигается
- На Windows могут понадобиться права администратора
- Проверьте что FAILSAFE не срабатывает

### TTS не говорит
- Установите голоса из системы
- На Windows: Панель управления -> Речь

```python
# В main.py
def _execute_agent_action(self, action):
    if action.get("type") == "new_action":
        # выполнить действие
```
